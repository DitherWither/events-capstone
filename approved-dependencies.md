# Approved Dependencies

This document outlines the approved dependencies for the project, ensuring that all libraries and tools used are vetted for security, performance, and compatibility with our technology stack.

- PostgreSQL
- Drizzle ORM
- Next.js (App Router)
- Arktype
- Tailwind CSS
- Any Tanstack libraries (e.g., React Query, React Table)
- React Hook Form
- Any library pulled by shadcn/ui
- Resend API
